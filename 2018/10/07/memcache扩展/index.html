2. 安装memcached

2.1 创建memcached用户

2.2 解压源码包并编译安装

#tar zxvf memcached-1.4.24.tar.gz

#cd memcached-1.4.24

#./configure –prefix=/usr/local/memcached

#make && make install

2.3 添加service脚本

#vi /etc/rc.d/init.d/memcached
